# olist-ecommerce-sql-analysis
SQL analysis of Olist E-Commerce dataset using PostgreSQL.

Olist E-Commerce Sales Analysis

Project Overview - This project analyzes the Olist Brazilian E-Commerce dataset using PostgreSQL. The objective is to derive business insights related to customer behavior, sales performance, product trends, payment methods, and revenue generation.

Dataset Source: Olist Brazilian E-Commerce Dataset

Tables Used:
- customers
- orders
- order_items
- product_database 
- order_payment

SQL Concepts Used
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- CASE WHEN
- Common Table Expressions (CTEs)
- Subqueries
- RANK()
- DENSE_RANK()
- ROW_NUMBER()
- LAG()
- LEAD()

 Business Questions Solved
- Revenue Analysis
- Customer Analysis
- Product Analysis
- Order Analysis
- Payment Analysis
- State-wise Performance Analysis

Tools Used
- PostgreSQL
- pgAdmin 4
- GitHub	

Project Outcome- Analyzed sales, customer purchasing behavior, product performance, and revenue trends through 24+ business-focused SQL queries.
